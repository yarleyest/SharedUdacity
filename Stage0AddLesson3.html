Getting Started with HTML

WWW - Basics of the web

  • Web: Collection of HTML documents
  • HTML (Hypertext Market Language): Main type of document on the web - Basic language for almost every web page
  • Hyperlinks (links): Connections between HTML documents/pages

Components of the web

A - Browser
A program that runs on your computer to display files found on the web
B - Internet:
The world's largest computer network
C - HTTP:
Main protocol on the web
D - Servers:
Computers that host the files that make up the web

More about HTML

Is made up of:

  • text content
  • markup (what the context looks like)
  • reference to others documents
  • links to other pages

**Tags**

HTML Markup is made of "tags"
  • Typical structure of tag: <Name> Content </Name>
  • All the structure is called Element
  • Tag Parts: Open tag (<Name>) Closing tag (</Name>)
  • Fundamental Difference between open and closing tag: Slash in the closing tag

Bold tag

 Structure: <b> content </b>

Italic tag

 Structure: <em> content </em>

HTML Attributes

Tags have and use multiple attributes.
Structure to include an attribute:
<tag attr="value"> content </tag>

Example: Anchor tag structure.
Anchor tag is for making links <a href="www.somepage.com"> this the page </a>
href is the attribute name
www.somepage.com is the value of the attribute

Image tag (for include images)

 Structure with some attributes: <img scr="url" alt="text" >
Image is a tag without closing tag. This kind of tags are called "void tags" because they have not content

Whitespace - br tag

 It's a void and inline tag
Structure: <br>

Paragraph (p tag)

 Structure: <p> content </p>

Inline/Block elements

Block- create an invisible box around the text with height and widht
Examples of block elements: <p> <form> <div>

Inline - act as a return
Examples of inline elements: <br> <em> <img> <strong> <a>

Container Elements

Attribute class in span and div comes from CSS Languaje

Span tag

(Inline element) Examples
Structure:
  <span> content </span>		   <span class="foo"> content </span>

Div tag

(Block element) Examples
Structure:
  <div> content </div> 		  <div class="bar"> content </div>

HTML Documents Structure

Typical form

<!DOCTYPE HTML>
<html>
  <head>
    <title> Title </title>
    <body>
Content
    </body>
  </head>
</html>

Explanation of the basic parts of HTML document:

<!DOCTYPE HTML> (Document type)
<html> (Opening tag html - surrounds the entire document)
<head> (Meta-data important for JavaScript and CSS) (opening tag)
<title>Title</title> (Title of web page - appaer in the top browser window o in browser tab)
<body>Content</body> (Content of web page)
</head> (closing tag head)
</html> (closing tag html)

HTML and CSS - are both languages -

  • "DOM" (Document Object Model) - the tree-like structure of a page
  • DOM is a standard convention for representing and interacting with elements in HTML
  • The nodes of every document are organized in a tree structure, called the DOM tree
  • A cross-platform and language-independent convention for representing and interacting with objects in HTML (and other markup languages).
  • HTML controls the "structure" of a web page
  • Basic word in HTML is a tag
  • Browser turns HTML tags into elements that form a tree
  • Any element is defined by tags (starting tag and ending tag)
    <tag> Content </tag>
    (Content could be text, empty or other element)
    • Any tag has a type and atributes with values
    Example:<h1 class="article"> content </h1>
  • CSS controls the "style" of a page (how it looks)
  • Cascading Style Sheets - CSS - style sheet language used for describing the look and formatting of a document written in a markup language
  • Specific syntax and rules to change how elements look on the page
